Why This Is The Reality

VS Code extensions are UI plugins, not CLI tools. They: - Run in the renderer process (not headless) - Require user interaction for security - Cannot be scripted via command line - Are designed for interactive use

The yzane extension works great for batch export, but you must trigger it through the UI.
