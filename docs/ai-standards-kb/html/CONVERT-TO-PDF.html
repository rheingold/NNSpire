Converting Markdown to PDF

You have 7 markdown documentation files ready to convert to PDF. Choose your preferred method below.

Method 1: VS Code Extension (Recommended)

Easiest option - Convert files directly in VS Code.

Installation

  1. Open VS Code
  2. Go to Extensions (Ctrl+Shift+X)
  3. Search for “markdown-pdf” by yzane
  4. Click Install

Usage

For each markdown file: 1. Open the file in VS Code 2. Right-click in the editor 3. Select “Markdown PDF: Export” (or use Ctrl+Shift+P and search for it) 4. Choose output location

The PDFs will be generated in the pdf/ directory structure.

Method 2: Pandoc Command Line

Best for batch conversion - Convert all files at once.

Installation

Download and install Pandoc from: https://pandoc.org/installing.html

Batch Conversion Commands

Run these commands from the workspace directory:

# Create output directories
New-Item -ItemType Directory -Path "pdf/standards" -Force
New-Item -ItemType Directory -Path "pdf/annexes" -Force

# Convert INDEX.md
pandoc INDEX.md -o pdf/INDEX.pdf --pdf-engine=xelatex --toc

# Convert all standard files
Get-ChildItem -Path "standards/*.md" | ForEach-Object {
    $name = $_.BaseName
    pandoc $_.FullName -o "pdf/standards/$name.pdf" --pdf-engine=xelatex --toc
}

# Convert all annex files
Get-ChildItem -Path "annexes/*.md" | ForEach-Object {
    $name = $_.BaseName
    pandoc $_.FullName -o "pdf/annexes/$name.pdf" --pdf-engine=xelatex --toc
}

Troubleshooting

Pandoc Not Found (Windows)

If you see “Pandoc not found” after installation: 1. Restart your terminal or PowerShell 2. Restart VS Code if using the extension 3. Verify installation: pandoc --version

PDF Rendering Issues

If PDFs don’t render formulas or code blocks correctly: - Make sure you’re using --pdf-engine=xelatex (requires LaTeX) - Alternative: use --pdf-engine=weasyprint or --pdf-engine=wkhtmltopdf

Cross-Links Not Working

PDF files support internal cross-links. If they don’t work: - Use --pdf-engine=xelatex for best compatibility - Open PDFs in Adobe Reader for full link support
